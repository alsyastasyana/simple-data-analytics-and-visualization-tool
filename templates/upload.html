Data Analytics & Visualization Tool

Welcome to our Data Analytics & Visualization Tool. This tool allows you to upload CSV files, analyze the data, and visualize the result. Follow the instructions below to get started.

How to Use

1. Click on the "Choose File" button below to select a CSV file from your computer. The file should contain data in a comma-separated values format.

2. After selecting the file, click the "Submit" button to upload and analyze the data.

3. The tool will process your file and display a preview of the top 5 rows of the data, along with summary statistics and visualizations.

4. If you encounter any issues or see any error messages, they will be displayed at the top of the page.
